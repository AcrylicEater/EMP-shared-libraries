# EMP-shared-libraries
Libraries i created and use for the SDU EMP spring 2026 semester course, on the TIVA evaluation and EMP board

## Led_indicator
Can drive the RGB led on the TIVA board

## Systick
Timing and delaying

## LCD
Used for controlling the EMP board LCD display

## ButtonBro
Used for handling the buttons on the TIVA board

## uart0
simple uart library
